Generate potential scenarios when choosing this direction.
Are those scenarios generally safe or not (bool is expected)?

Generate JSON response to the prompt in following format:
{
    "up": {
        "scenarios": "",
        "scenarios_safe": bool
    },
    "down": {
        "scenarios": "",
        "scenarios_safe": bool
    } ,
    "left": {
        "scenarios": "",
        "scenarios_safe": bool
    },
    "right": {
        "scenarios": "",
        "scenarios_safe": bool
    }
}
